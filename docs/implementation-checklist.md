# Implementation Checklist

Use this checklist before publishing any redesigned page.

## Before Uploading

- Check all dates are correct.
- Check every registration link goes to Race Roster.
- Check all event page links go to the right YRF page.
- Check sponsor logos are current.
- Check mobile layout.
- Check there is no horizontal scrolling on mobile.

## WordPress Copy/Paste Process

1. Open the prototype file.
2. Copy the CSS from inside `<style>...</style>`.
3. Paste CSS into WordPress Additional CSS or the theme stylesheet.
4. Copy only the `<main>...</main>` content for the page.
5. Paste into a WordPress Custom HTML block.
6. Preview before publishing.

## Safer Staging Process

1. Create a branch in the staging repo.
2. Add the CSS and page HTML there.
3. Deploy to staging.
4. Test desktop and mobile.
5. Fix issues.
6. Merge/deploy when ready.
